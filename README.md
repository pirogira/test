# Тестовое задание: адаптивная страница + админка

Реализован проект на Flask:

- публичная адаптивная страница;
- админка для редактирования текстов и изображений;
- изменение порядка блоков страницы (drag-and-drop + вверх/вниз);
- редактирование SEO `meta description`.

## Запуск локально

1. Установить зависимости:

```bash
py -m pip install -r requirements.txt
```

2. Запустить приложение:

```bash
py app.py
```

3. Открыть в браузере:

- сайт: `http://127.0.0.1:5000/`
- админка: `http://127.0.0.1:5000/admin/login`

## Доступ в админку

- Пароль по умолчанию: `admin123`
- Можно переопределить через переменную окружения `ADMIN_PASSWORD`.

## Что можно редактировать в админке

- `page title` и `meta description`;
- заголовок, текст, кнопку блока;
- изображение (URL или загрузка файла);
- включение/выключение блока;
- порядок блоков на странице.
